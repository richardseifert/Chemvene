# Chemvene  
[![Build Status](https://travis-ci.com/richardseifert/Chemvene.svg?branch=master)](https://travis-ci.com/richardseifert/Chemvene)

A convenient <ins>Che</ins>mical <ins>M</ins>odel <ins>V</ins>isualization and analysis <ins>EN</ins>gin<ins>E</ins> for processing output from the Michigan protoplanetary disk chemical kinetics code.

For an overview of features, detailed installation and setup, and full documentation, please take a look at the [wiki](https://github.com/richardseifert/Chemvene/wiki) for this repository!

## Installation

Chemvene can be installed with pip, or by cloning this repository.

### 1. Installation with pip

```
pip install [--user] Chemvene
```

### 3. Local install

```
git clone https://github.com/richardseifert/Chemvene.git
cd Chemvene
pip install [--user] .
```
